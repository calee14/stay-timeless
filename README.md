# stay-timeless

To install dependencies:

```bash
bun install
```

To run:

```bash
cd mobile

# Side load app to mobile 
# device (sim or real).
bunx expo run:ios --device

# Start Metro which will 
# hot reload app updates 
# live in dev 
bunx expo start --dev-client

cd server
# Start server for mobile app
bun run dev
```
